Rojak Web
===========================

### Stack:

1. React
2. Redux
3. UIKit CSS
4. Ionicons

### Features:

1. Hot Reload
2. ES6 (`react, es2016, stage-2`)

### Development:

1. `npm install`
2. `npm start`
3. Open `http://localhost:3000`.

### Build

1. `npm run build`

### Deploy

1. `npm run build`
2. `npm run deploy`
3. Then `build` folder should be here `rojak-web.surge.sh`.
